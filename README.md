# README

## アプリケーションの概要
* 自分がどんなゲームや漫画、映画を見たのか（やった）、もしくは見たい（やりたい）のかを管理してくれるアプリです。
* todo doing doneの３つのステータスから選択して、分類することができ、積ん読もしくは積みゲーなどの助けになるアプリです。

## アプリケーションの機能一覧
* コンテンツ追加機能
* コンテンツ詳細確認機能
* コンテンツ編集機能
* コンテンツ削除機能
* ステータス色分け機能
* 検索機能
* ソート機能（ステータスやタイトル、年代順などそれぞれの項目ごとにソートできます）
* ページネーション機能
* 単体テスト
